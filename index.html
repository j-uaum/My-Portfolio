Folha de pagamento


Esse é um projetinho feito como exercicio de Php, porem foi migrado para Javascript. Ele tem interface de formulario de captura os dados e utiliza dentro do sistema, ele calcula salario bruto, liquido e taxas de IR, Inss e FGTS

Formulário de cadastro de alunos

Esse exercício é um formulário feito durante uma aula de PhP, se propõe a ser um formulário para coleta de dados e verificação, posteriormente será postada a segunda parte com o código PhP

Média de alunos

Esse exercício foi feito em Javascript, ele se sujeita a ser um sistema para calcular a média de alunos fictícios, usei várias formas de resolução, a minha favorita foi com o método “new” para criar um novo objeto. Esse projeto tem 5 versões, sendo a 5 a mais aprimorada

Dynamic Memories

Gas x Etanol

Sistema para cálculo de custo-benefício dos combustíveis etanol x gasolina. Esse site é um exercício criado por mim mesmo para exercitar meu código. Ele já utiliza o mesmo código que ele site, excluindo diversas páginas HTML e simplificando em apenas uma

Contador

Esse sistema é feito para ser um contador com parâmetro de início, fim e passo. Feito em javascript, é um exercício bem simples para treinar uma coisa tão básica

  • Tenho 9 projetos expostos no github e mais de 20 outros guardados e mais 5 em desenvolvimento. Tenho 2 anos de experiência séria com a linguagem


    • HTML5 & CSS3


    Essas são minhas aliadas no desenvolvimento com Js e PhP, nem tem como fugir delas. Esse site é feito 80% só de HTML e CSS e minha experiência também pode ser comprovada pelos meus projetos no Github, apesar de estarem um pouco ultrapassados.
